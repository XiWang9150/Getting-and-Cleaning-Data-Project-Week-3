#### Please follow the steps below to run and test `run_analysis.R`  
1. Download and save the data for project to your work directory.  
2. Unzip the zip file and copy all data from unzipped folder to your current work folder.  
3. Create another folder in your current work directory to save run_analysis.R.  
4. Run run_analysis.R  
5. You will obtain two files    
**`tidy_data_1.txt`** : This is the full tidy data with `Activity Label` and `Subject`  
**`tidy_data_2.txt`** : This is the second data with the average of each variable for each activity and each subject.
